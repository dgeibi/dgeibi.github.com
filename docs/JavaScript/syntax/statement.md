## if

```javascript
if (conditon) {
    statement1
} else {
    statement2
}
```

## do-while

```javascript
do {
    statement
} while (expression);
```

## while

```javascript
while(expression) {
    statement
}
```

## for

```javascript
for(initialization; expresion; post-loop-expression) {
    statement
}
```

## for-in

```javascript
for(property in expression) {
    statement
}
```

## switch

```javascript
switch (expression) {
    case value:
        statement
        break;
    case value:
        statement
        break;
    default:
        statement
}
```

```javascript
switch (true) {
    case condition:
        statement
        break;
    case condition:
        statement
        break;
    default:
        statement
}
```
